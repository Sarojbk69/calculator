# Scientific Calculator

A responsive scientific calculator designed specifically for science students. This calculator provides a comprehensive set of mathematical functions commonly used in scientific calculations.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, division
- **Scientific Functions**:
  - Trigonometric functions (sin, cos, tan)
  - Inverse trigonometric functions
  - Logarithmic functions (log, ln)
  - Exponential functions
  - Square root and power functions
  - Constants (π, e)
  - Factorial calculation
  - Percentage calculation
- **Memory Functions**: Store, recall, add to memory, subtract from memory, clear memory
- **Additional Features**:
  - Degree/Radian mode toggle
  - Dark/Light theme toggle
  - Calculation history display
  - Previous answer recall
  - Error handling

## Usage

1. Open `calculator.html` in a web browser
2. Use the calculator interface to perform calculations:
   - Click number buttons to input values
   - Click function buttons to perform operations
   - View results in the display area
   - Toggle between degree and radian modes for trigonometric calculations
   - Use the theme toggle to switch between light and dark themes

## Responsive Design

The calculator is fully responsive and works on:
- Desktop computers
- Tablets
- Mobile phones

The layout automatically adjusts based on the screen size to ensure optimal usability on all devices.
